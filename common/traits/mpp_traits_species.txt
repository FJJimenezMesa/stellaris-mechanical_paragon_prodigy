# ================================
# Variables base para traits de especies MPP
# ================================
@base_ai_weight = 0              		# Peso base AI (IA le asigna al rasgo al generar o seleccionar especies automáticamente.)
@default_ai_weight = @base_ai_weight  	# Peso AI por defecto
@default_crossbreeding = 1.0      		# Probabilidad de que esta especie pueda cruzarse con otras especies compatibles (1.0 = 100%)


# ================================
# 🧬 Species Trait: mpp_trait_nano_vector
# "Infección de nanorobots"
# ================================
# Descripción narrativa + lore:
#   Un vector biológico que transporta y mantiene nanorobots en la sangre. Estos nanites, conocidos como "sangrenaníticos", fueron originalmente desarrollados y liberados por el líder Parangón para asegurar su legado genético y capacidad de influencia incluso a distancia.
#   Los individuos portadores de este trait poseen longevidad mejorada, resistencia natural y habilidades sinérgicas para la producción y la investigación.
#   Su sangre se convierte en un conducto activo, un canal vivo que conecta su biología con la red de nanites del Parangón. El flujo de nanorobots es la base de una misteriosa herencia que impulsa la evolución de la especie hacia futuros estados biológicos avanzados.
# Descripción:
#   Un vector biológico que transporta y mantiene nanorobots en la sangre.
#   Funciona como portador activo de nanites ("sangrenanítica").
# Aplicación: Especies BIOLOGICAL, con posibilidad de cruzamiento avanzado.
# Enfoque: Rareza, longevidad, resiliencia y sinergias productivas.
# Afecta a: Pops y líderes biológicos.
# ================================
mpp_trait_nano_vector = {
    cost = 5
	icon = "gfx/interface/icons/traits/mpp_trait_nano_vector.dds"
	
	ai_weight = { @default_ai_weight }

	allowed_archetypes = { BIOLOGICAL }
    allowed_leader_archetypes = { BIOLOGICAL }

    potential = {
        is_biological = yes
    }

	species_potential_add = {
		can_do_advanced_gene_modding = yes
	}
	potential_crossbreeding_chance = @default_crossbreeding

    modifier = {
		leader_age = 50

    	mpp_species_leader_bonus = yes
		mpp_bio_population_bonus = yes
		mpp_job_production_bonus = yes
		mpp_planet_research_production_bonus = yes
    }
}

# ================================
# 🧬 Species Trait: mpp_parangon_nano_operancy
# "Parangón Nano-Infusión"
# ================================
Descripción narrativa + lore:
Rasgo exclusivo para robots, los cuales carecen de sangre y no pueden establecer control directo sobre los nanorobots. En cambio, la infusión de nanites se integra parcialmente en sus sistemas, proporcionando aumentos de producción, eficiencia laboral y capacidad de investigación.
Aunque la conexión es limitada, los nanites mantienen el legado del Parangón, sirviendo como un vínculo misterioso que permite a los robots recibir parte de su poder. La infusión nunca alcanza la simbiosis total de las máquinas, pero confiere una ventaja tangible y una presencia narrativa de la influencia del Parangón sobre toda la red mecánica del universo.
# Descripción:
#   Rasgo mecánico avanzado que mejora producción, felicidad, eficiencia de líderes y trabajos,
#   además de mejorar ensamblaje y definir costes de esclavos.
# Aplicación: Exclusivo para especies ROBOT
# Enfoque: Optimización mecánica, colonización eficiente y supremacía tecnológica.
# Afecta a: Pops y líderes robóticos.
# ================================
mpp_parangon_nano_operancy = {
    cost = 10
    icon = "gfx/interface/icons/traits/mpp_parangon_nano_operancy.dds"

	ai_weight = { @default_ai_weight }

    allowed_archetypes = { ROBOT }

    potential = {
        is_robot = yes
    }

    slave_cost = {
        energy = 750
    }

    assembly_score = {
        modifier = {
            add = 1.5
            from = { has_mining_designation = yes }
        }
        modifier = {
            add = 0.5
            from = { has_rural_designation = yes }
        }
    }

    modifier = {
		mpp_species_leader_bonus = yes
		mpp_robotic_population_bonus = yes
		mpp_job_production_bonus = yes
		mpp_research_production_bonus = yes
    }
}
